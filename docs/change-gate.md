# Change Gate

Минимальный протокол «Pressure → Surface Check → Proof → Patch», типы давления (Bug/Feature/Debt/Ops), требования к изменениям [FROZEN].  
Формат (в PR/Issue/Plan):  
- Intent: <one sentence>  
- Pressure: Bug | Feature | Debt | Ops  
- Surface impact: (none) | touches: <Surface item(s)> [FROZEN/FLUID]  
- Proof: tests: <files or new tests>  
Если touches [FROZEN] — приложите Migration/Compatibility note (Old→New, стратегия, окно/версия).
